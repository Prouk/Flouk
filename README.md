
# Flouk

Flouk is my default environment flake.

![alt text](https://github.com/Prouk/Flouk/blob/main/screenshot.png)

## Features

- NixOS / Home-manager Configs
- Hyprland / Rofi / Wayland
- Zen-browser updated flake (will update to use package when it will be available)
- Zed-editor as default text editor (cause f*ck vscode)
- Steam

## Usage/Examples

Dont' forget to edit the host / username in the flake.nix then the hardware-configuration.nix and disk.nix to match your setup.

Then, run the flouk.sh file:

```sh
[host@hostname:~/Flouk]$ ./flouk.sh
```
